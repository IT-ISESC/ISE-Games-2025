## Getting Started

### Project Structure

```text
├── public/
│   └── fonts/
│       └── Get fonts here
│   └── images/
│       └── Add or use images here (ideally use webp)
├── src/
│   └── components/
│       └── Add components here in '.astro' files
│   └── pages/
│       └── api/
│       │   └── Write APIs here
│       └── Add pages here in '.astro' files
└── package.json
```

### Using pnpm commands
 - `pnpm i` => Install dependencies
 - `pnpm run dev` => Run dev server

### Tech Stack
 - Astro
 - Tailwind
 - Firebase
 - Prettier
